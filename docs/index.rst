``cita-sdk-python`` 是 `CITA高性能区块链 <https://www.citahub.com>`_ 的Python binding, 提供下述功能:

- :class:`~cita.CitaClient` 类封装了CITA JSON RPC的调用.
- :class:`~cita.ContractClass` 类封装了合约文件和ABI. 使用Python的语法来完成合约部署.
- :class:`~cita.ContractProxy` 类封装了已部署的合约. 使得对合约方法的调用看起来就像对Python Object方法的调用.

请先阅读 :ref:`基本用法`, 更多细节可以查阅 :ref:`References` .
